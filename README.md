# Hello World Collection

一个包含多种编程语言 "Hello, World!" 示例的集合项目，适合编程初学者快速上手或作为参考模板。

## 📁 项目结构

```
.
├── hello.c          # C 语言示例
├── hello.cpp        # C++ 示例
├── hello.go         # Go 语言示例
├── Hello.java       # Java 示例
├── hello.py         # Python 示例
└── README.md        # 项目说明文档
```

## 🚀 快速开始

### C 语言
```bash
gcc hello.c -o hello
./hello
```

### C++
```bash
g++ hello.cpp -o hello
./hello
```

### Go
```bash
go run hello.go
```

### Java
```bash
javac Hello.java
java Hello
```

### Python
```bash
python hello.py
```

## 💡 输出示例
所有程序运行后都将输出：
```
Hello, World!
```

## 📝 说明
- 每个文件都是独立的可运行示例
- 代码遵循各语言的最佳实践和简洁风格
- 适合作为学习新语言的第一个程序参考
